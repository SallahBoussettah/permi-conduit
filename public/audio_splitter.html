Audio File Splitter for Uploads

Split large audio files into small chunks that can bypass PHP upload size limits

Why This Tool Exists

The server is currently configured with a 2MB upload limit. This tool helps you split larger audio files into smaller chunks that can be uploaded individually to bypass this limitation.

How It Works

Select your audio file - Any audio file of any size can be processed
Split the file - Your file will be split into 2MB chunks (or smaller if you prefer)
Download each chunk - Save all the chunks to your computer
Upload them in sequence - Upload each chunk as a separate course material, numbered in sequence

Privacy Note: All processing happens in your browser - no files are uploaded during the splitting process.
